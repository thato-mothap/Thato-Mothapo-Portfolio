# Thato Mothapo — Personal Portfolio

A personal portfolio website built from scratch and deployed via **GitHub → Netlify**.

Live site: [https://thatomothapo.netlify.app](https://thatomothapo.netlify.app) 

---

## About

This is the source code for my personal portfolio — designed and coded entirely by hand with no frameworks or templates. It showcases my background, projects, experience, and certifications as a BSc Computer Science graduate from Eduvos Midrand.

Built for potential employers, collaborators, and clients to get a clear picture of who I am and what I can build.

---

## Features

- **Responsive design** — works across desktop, tablet, and mobile
- **Dark / light mode** — respects system preference, persisted via localStorage
- **Scroll-reveal animations** — elements animate in as the user scrolls
- **Morphing profile image** — CSS keyframe animation on the profile frame
- **Custom cursor** — animated dot + ring cursor (desktop only)
- **Animated skill bars** — proficiency bars triggered on scroll
- **Contact form** — opens the user's mail client via `mailto:`
- **Netlify deployment** — auto-deploys on every push to the `main` branch

## Deploying

The site is connected to Netlify via this GitHub repository. Any push to the `main` branch triggers an automatic redeploy — no manual steps needed.

To run locally, simply open `index.html` in a browser. There is no build step or package manager required.
